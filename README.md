# Combined Events Calculator

Mobile app for calculating combined‑events points (Decathlon, Heptathlon, Pentathlon) built with React Native + Expo.

## Features

- Men’s Decathlon, Men’s Heptathlon, Women’s Heptathlon, Women’s Pentathlon
- Real‑time scoring with validation
- Time and field input formatting
- Total score and World Athletics comparison

## Quick start

```bash
git clone [repository-url]
cd CombinedEventsApp
npm install
npx expo start
```

Run on device via Expo Go (scan QR), or use an emulator/simulator.

## Input format

- Time: `10.23` or `2:13.60`
- Field: `7.25`
